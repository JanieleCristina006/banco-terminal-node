# Banco Terminal - Projeto Node.js

Este é um projeto simples de backend em Node.js que simula operações básicas de um banco no terminal, criado como projeto inicial de aprendizado em backend.

## Funcionalidades

- Criar conta bancária com nome único
- Consultar saldo da conta
- Realizar depósitos
- Realizar saques
- Transferência entre contas
- Interface de linha de comando interativa com Inquirer.js
- Mensagens coloridas para melhor visualização com Chalk
- Armazenamento de dados em arquivos JSON

## Tecnologias Utilizadas

- Node.js
- Inquirer.js para prompts no terminal
- Chalk para estilização de texto no console
- Sistema de arquivos (fs) para leitura e escrita dos dados

## Como rodar

1. Clone este repositório  
2. Execute `npm install` para instalar as dependências  
3. Execute `node index.js` para iniciar o programa  
4. Siga as instruções no prompt para utilizar as funções do banco

## Estrutura do Projeto

- `index.js` - Arquivo principal com toda a lógica da aplicação  
- `contas/` - Pasta onde ficam armazenados os arquivos JSON das contas

## Aprendizados

Este projeto me ajudou a entender conceitos importantes de backend como:

- Manipulação de arquivos com Node.js  
- Criação de interfaces interativas no terminal  
- Controle de fluxo de dados e lógica de negócio básica  
- Boas práticas com tratamento de erros e validação de dados
